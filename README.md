# A Mad Tea Party

A mobile-first party invitation built as a tiny top-down pixel RPG. Play as Alice
through an abridged Wonderland — shrinking and growing through size-gated doors,
gathering friends who follow along in a line — until the journey ends at the Mad
Tea Party, where setting the table reveals the invitation.

▶ **Play:** https://jpb33333.github.io/mad-tea-party/

This repository contains only the compiled, playable build.

## Disclaimer

A noncommercial fan work based on Lewis Carroll's public-domain novels
*Alice's Adventures in Wonderland* (1865) and *Through the Looking-Glass* (1871),
with visual reference to John Tenniel's public-domain illustrations.
